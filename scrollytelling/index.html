What is Parallax?

Parallax scrolling is a web design technique where background images move past the camera more slowly than foreground images, creating an illusion of depth in a 2D scene and adding to the immersion. This simple example uses the CSS `background-attachment: fixed;` property to achieve the effect.

A 3D Illusion

How It Works

The magic comes from `background-attachment: fixed;`. This CSS rule tells the browser to keep the background image's position fixed relative to the viewport. As you scroll, the content blocks (like this one) move normally, but the background image stays in place, revealing more of itself.
